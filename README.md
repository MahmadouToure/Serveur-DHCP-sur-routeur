# Serveur-DHCP-sur-routeur

## Description 
Projet réalisé sur Cisco Packet Tracer qui simule un serveur DHCP sur un routeur

## Équipements utilisés
1 Router 2911    
3 PC     

## Configuration principale
Création du pool DHCP     
Configuration manuelle de IP du routeur    
Adressage automatique des IP hôtes finaux   
Test de connectivité
